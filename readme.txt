== Description ==
Theme Powerkit is WordPress free plugin with multiple feature. Plugin have 5 useful widget like Author, Category, Recent Posts, Social Icon and Tab Posts Widgets and also, this plugin have webmaster verification meta field, Open Graph, Twitter Summery Card and you can add script before header and after footer as well.

== Plugin Features: ==
* <strong>Author, Category,Recent Posts, Social Icon and Tab Posts Widget</strong>
* <strong>Webmasters Verification meta</strong>
* <strong>Open Graph</strong>
* <strong>Twitter Summary Card</strong>
* <strong>Header and Footer Script Area</strong>

== Installation ==
	= Using The WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Search for Theme Powerkit
	* Click Install Now
	* Activate the plugin on the Plugin dashboard

	= Uploading in WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Navigate to the 'Upload' area
	* Select theme-powerkit.zip from your computer
	* Click 'Install Now'
	* Activate the plugin in the Plugin dashboard

	= Using FTP =
	* Download theme-powerkit.zip
	* Extract the theme-powerkit directory to your computer
	* Upload the theme-powerkit directory to the /wp-content/plugins/ directory
	* Activate the plugin in the Plugin dashboard

	= Setting Options =
	* Setting Page is located as Theme Powerkit on your Dashboard
	
	= Permalink Setup =
	* Go to Settings -> Permalink and click on "Save Changes" if your custom post type redirects you to 404 page.
